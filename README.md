🎲 Dice Game (Python)

A simple and interactive Dice Game built using Python where you compete against the computer in a Best of 5 match.

The first player to win 3 rounds wins the game!

🚀 Features
🎯 Best of 5 gameplay
🎲 Random dice rolling
⏳ Rolling animation effect
🖥️ ASCII dice face display
🤖 Play against computer
🔁 Replay option
📊 Live score tracking
🛠️ Technologies Used
Python 3
Built-in modules:
random
time
📂 Project Structure
dice_game.py   # Main game file
README.md      # Project documentation
▶️ How to Run
Install Python (if not installed)
Clone or download this repository
Run the program:
python dice_game.py
🎮 How to Play
Press ENTER to roll your dice
Both you and the computer will roll a dice 🎲
The higher number wins the round
First to win 3 rounds wins the match
If both rolls are equal → it's a tie (no points)
